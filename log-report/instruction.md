There is an Apache-style access log located at /app/access.log.

Analyze the log and write a JSON report to:

/app/report.json

The JSON report must satisfy all of the following:

1. Include the total number of requests.
2. Include the number of unique client IP addresses.
3. Include the most frequently requested path.
4. Produce valid JSON at the exact output path /app/report.json.
